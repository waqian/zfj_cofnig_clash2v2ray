# Clash 配置转换工具

这是一个针对 [https://zfj.so](https://zfj.so) 的 Clash 配置转换工具。该工具可以将 Zfj 网站提供的订阅链接转换为适用于 Clash 的配置文件。

## 功能

- 将 Zfj 网站提供的订阅链接（例如： `https://zfj.so/xxxx`）转换为 Clash 配置文件。
- 允许用户自定义配置地址（需要手动修改配置文件中的链接）。
- 简化 Clash 配置生成的过程，提高配置管理效率。
